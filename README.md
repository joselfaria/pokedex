# Pokédex (Next.js)

Este repositório contém o desenvolvimento de uma Pokédex básica construída com **Next.js**, **TypeScript** e **TailwindCSS**, usando a [PokéAPI](https://pokeapi.co/) pública.

O projeto está em desenvolvimento ativo e serve como estudo das ferramentas utilizadas.

## 🚀 Objetivo do Projeto

O objetivo é construir uma Pokédex funcional ao longo dos estudos.


## 📂 Estrutura do Projeto

- `app/`: Diretório principal do App Router do Next.js, contendo as páginas e layouts.
- `components/`: Componentes reutilizáveis da interface, como cards e barra de busca.

## 🛠️ Funcionalidades Atuais

- **Listagem de Pokémons**: Exibe os pokémons da primeira geracao.

## 🔜 Funcionalidades Planejadas

- Busca por nome
- Página de detalhes de cada pokémon
- Filtro por tipo
- Paginação

## 🏃 Como Executar

**Pré-requisitos:** Node.js 18+

Para rodar o projeto localmente:

```bash
# Clone o repositório
git clone https://github.com/joselfaria/pokedex.git

# Acesse a pasta
cd pokedex

# Instale as dependências
npm install

# Rode o servidor de desenvolvimento
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no navegador.
